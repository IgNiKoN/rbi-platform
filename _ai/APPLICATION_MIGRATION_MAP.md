# Application Migration Map

> Дата: 2026-07-05 (архитектор). Связанный файл: `_ai/FILE_MIGRATION_MAP.md` (построчная классификация файлов).

## Goal

Полный переезд текущего RBI Quality Pro в RBI Platform без потери функционала. RBI Platform — не новая оболочка рядом со старым приложением, а целевая архитектура, в которую **полностью** переезжает текущее приложение. Ничего не остаётся постоянным legacy-блоком внутри платформы.
